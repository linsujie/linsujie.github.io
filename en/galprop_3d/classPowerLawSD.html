◆ PowerLawSD()

PowerLawSD::PowerLawSD ( const utl::Parameters &  pars)

Construct a new broken power law spectral distribution

The class supports arbitrary number of breaks and the variable used in the power law can be set with

  • powerLawType = ETOT | EKIN | PTOT | PNUC | RIGIDITY | GAMMA | BETA with the key defined as
  • ETOT : total energy
  • EKIN : kinetic energy per nucleon [ (ETOT-m0)/A ]
  • PTOT : total momentum
  • PNUC : momentum per nucleon [ PTOT/A ]
  • RIGIDITY : rigidity [ PTOT/|Z| ]
  • GAMMA : Lorentz factor [ ETOT/m0 ]
  • BETA : speed in units of c

Assumes the double vector spectral parameter has the format <index> [<break>, <index>]* where the first index is mandatory and the breaks must be given in ascending order (smallest first)

This class follows the convention of negative indices, i.e. F(x) = x**(-index)
