Helper functions to calculate intersections with a cylinder and a box

Returns the distance along the line of sight for the pair of points intersecting the shape l=0 points in the negative x direction, b=pi/2 points in the z direction So shifted Galactic coordinates assuming the sun is at positive x First part of pair is always smaller of the intersection points. If camera is within shape, return 0 for the first coordinate Returns negative values in the pair if no intersection can be found (-1,-1)
